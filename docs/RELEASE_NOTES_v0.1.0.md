# WTmap v0.1.0

首个可公开安装版本，面向 AYN Thor 双屏掌机。

## 功能

- 通过《战争雷霆》`8111` 接口显示实时地图、单位、战区和机场信息。
- 陆战与空战界面自动切换，支持地图缩放和拖动。
- 空战姿态仪与精简飞行参数面板。
- 修正异常字母/数字地图图标、重生点重叠和地面单位边框。
- 红色闪烁火控标记。
- 自定义操作按钮、名称和键盘映射。
- “瞄准”模式在下屏显示上屏镜像，保留下屏控制栏，并支持下屏拖动同步控制上屏。
- IP 连接区和底部按钮栏均可独立折叠。

## 安装提示

- 在 `Assets` 中下载 `WTmap-v0.1.0.apk`。
- 详细步骤见仓库中的 [INSTALL.md](https://github.com/ChaselTH/WTmap/blob/main/INSTALL.md)。
- 地图功能需要 Thor 与运行游戏的电脑处于同一局域网。
- 硬件按键与瞄准镜像需要 Magisk/Root 授权。
- 本版本主要针对 AYN Thor 当前双屏系统验证。
